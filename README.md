# HACKITHON 2022 - Analýza pohybové aktivity z chytrých náramků

**Nazev** je webová aplikace zabívající se zpracováním dat z náramků značky xiaomi band 4.
Data se po zpracování zobrazí ve webovém rozhraní.

## Použité technologie
 - ### FRONTEND
   - HTML/CSS
   - JS
 - ### BACKEND
   - Python
     - Pandas
